# Archived — Tokyo Airbnb group project (MKTG 4550, Spring 2026)

Reference material only. **Not part of the site build; nothing here is published.**

## What this is
Team 5 ("Hulk") class project for MKTG 4550 (Prof. Hong), Seattle University.
Members per the slides: Clark Hayashi, Anthony Merlino, Keshav Dhawan, Blake Dorn, Ben Sakamoto.

- `Group 5. Hulk Airbnb Project - Midterm Presentation Slides.pptx` — midterm (May 7, 2026): Inside Airbnb Tokyo data (27,945 active / 25,209 modeled listings), price regression (log-log, adj. R² 0.514), ward/room-type/timing value analysis for post-grad travelers.
- `Group 5 Final Presentation.pdf` — final: PCA/factor analysis (5 factors, ~51% variance), hierarchical clustering (Cluster 3 = 74.1% of market, Cluster 4 = 15.6% premium), classification models in R (Random Forest AUC 0.821), review-text sentiment/topic analysis.

## Why it's archived, not published
Clark's decision (July 2026): the group deliverable doesn't meet the bar he wants
on the portfolio. He is rebuilding the analysis independently, end-to-end
(solo, Python + Tableau, host-facing framing). This archive exists so the
rebuild can reference the dataset source, benchmark the group's findings,
and stay honest about what was group work vs. solo work.
