# Source material — Hawai'i Airbnb vs. Hotels (BUAN 4220, Data Visualization)

**Status: liked, approved for the site.** These are the source files behind the
upcoming "Finding the Best Value Stays in Hawai'i" case study
(draft at `drafts/hawaii-value-stays.md`). Not part of the site build.

Team: Jayden Respicio & Clark Hayashi (two-person project).

- `BUAN 4220 Memo.docx` — findings memo. The six quantified takeaways
  (hotel $250 vs Airbnb $283 averages; 4.79/5 avg rating; 70% of sub-$1,000
  listings pricier than hotels; Maui/Kauai most expensive, Molokai lowest;
  Oahu ≈29% of listings; waterfront price premium).
- `BUAN4220_Project-1.twb` — the Tableau workbook: 14 worksheets + "Main"
  dashboard, `hawaii_data` source. **To do: publish to Clark's own Tableau

  (short-term rentals restricted outside hotel/resort zones; listings
  cluster where hotels are; 0 hotels in Central Oahu).
